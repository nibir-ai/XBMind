✨ Features

Everything you need,
nothing you don't

Fully offline, privacy-first smart speaker capabilities — no cloud, no subscription, no data leaving your network.

🔵

Bluetooth Auto-Pair

Discovers, pairs, and auto-reconnects with exponential backoff via D-Bus. Works with any A2DP Bluetooth speaker.

🎙️

Always-On Microphone

Low-latency audio capture via sounddevice with automatic USB microphone detection and configuration.

👂

Wake Word Detection

openWakeWord (ONNX) running in a dedicated OS thread. Default: "Hey Jarvis" — fully customizable.

🗣️

Speech-to-Text

Offline transcription via faster-whisper (CTranslate2) with optional Google Cloud STT fallback.

🧠

LLM Brain

Ollama for offline inference with swappable cloud providers: OpenAI, Claude, and Gemini support built in.

🔧

Built-in Tools

Weather, date/time, timers, Wikipedia lookups, and shell command execution — all via the LLM tool system.

📢

Text-to-Speech

Piper TTS for high-quality offline speech synthesis via subprocess pipe. Optional ElevenLabs fallback.

💾

Conversation Memory

SQLite-backed conversation history with automatic summarization for long-term context retention.

🏥

Health Check

HTTP endpoint at localhost:7070/health with JSON component status for monitoring and automation.

⚙️

Systemd Ready

User service with auto-start and journald logging. Enable, start, and forget — it just works.

🚀 Quickstart

Up and running in minutes

Three commands. That's all it takes.

1 Clone the repository

$ git clone https://github.com/nibir-ai/XBMind.git
$ cd XBMind

2 Run the installer

$ ./scripts/install.sh
# Interactive installer handles everything:
# system deps, Python venv, AI models, Bluetooth, systemd

3 Start XBMind

$ source .venv/bin/activate
$ python -m xbmind.main
# Or run as a systemd service:
$ systemctl --user enable --now xbmind
